Expand description

Callbacks fired at various stages of compilation.

The callbacks are called in the following order:

  1. after_parse
  2. after_lower
  3. after_codegen
  4. after_render_assembly
  5. after_compile

If a callback returns ControlFlow::Break, the compiler will stop running and return early.
